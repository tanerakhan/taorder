TaOrder — Windows Kurulum (Müşteri)

1. TaOrder-1.0.1-x64.exe dosyasını indirin.
2. Dosyaya çift tıklayın.
3. Kurulum sihirbazında "İleri" deyin, kurulumu tamamlayın.
4. Masaüstündeki "TaOrder" simgesine tıklayarak programı açın.

Windows güvenlik uyarısı çıkarsa:
  "Daha fazla bilgi" → "Yine de çalıştır"

=== GÜNCELLEME (eski sürüm yüklüyse) ===

- Kurulum mevcut sürümün üzerine yazılır.
- Menü, masa ve sipariş verileriniz silinmez; bilgisayarınızda saklanır.
- Veriler: Windows'ta %APPDATA%\taorder\taorder.db

=== YENİ KURULUM ===

- Program boş menü ve boş masa listesi ile açılır.
- Önce Kategoriler → Menü → Masalar sekmelerinden kendi verilerinizi girin.

Verileriniz bu bilgisayarda saklanır; internet gerekmez.

Destek için satıcınıza başvurun.
